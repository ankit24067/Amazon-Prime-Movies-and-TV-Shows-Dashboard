# Amazon-Prime-Movies-and-TV-Shows-Dashboard
My first Power Bi Dashboard

## Overview
This dashboard provides visual insights into the performance and distribution of content on Amazon Prime. It includes various data visualizations such as bar charts, pie charts, maps, and line graphs.

## Features
- **Total Titles**: Displays the total number of titles available on the platform.
- **Total Ratings**: Shows the distribution of ratings across all shows.
- **Genre Analysis**: Breaks down the number of shows by genre.
- **Director Analysis**: Highlights the number of titles by director.
- **Release Year Trends**: Shows trends in content release over the years.

## Data Source
The data used in this dashboard is sourced from Kaggle, which includes detailed information on movies and TV shows available on Amazon Prime.
